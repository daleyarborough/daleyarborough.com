    • For Example
      var listItems = $("header nav li"); >>> side note, $("header nav li"); is referred to as a "wrapped set"

      listItems.filter(":first").css("font-size", 18px");
      This is a good technique because it doesn't inefficiently perform a trivial task that could be easly taken care of with
      simple css, but because the execution is against the wrapped set (var listItems = $("header nav li")), it enables
      manipulation of the list items inside of the navbar that is inside of the header. And will query the wrapped set one line at a time.
